Step-by-step: Get Bot Token & Access Token

  1. Create Telegram bot

    Open Telegram and search BotFather. Send /newbot, set bot name and username.

  2. Copy bot token

    BotFather will return a token like 123456789:ABC.... This is your TeleRealm Access Token.

  3. Start chat with your bot

    Open your bot profile, press Start, and send a test message. If using group/channel, add bot and grant rights.

  4. Get Chat ID

    Use this URL in browser (replace {TOKEN}):

    https://api.telegram.org/bot{TOKEN}/getUpdates

    Look for chat.id in response JSON.

  5. Use in TeleRealm Upload Workspace

    Paste bot token into Bot Token, paste chat.id into Chat ID, then go to /upload and click Save Local Config.

  6. Upload and get secure link

    Select file, click Upload Files, then copy secure_url to share.

Security Note

Never expose your bot token in public frontend production apps. Use backend proxy or rotate token immediately if leaked.
