Local History

Uploaded files and secure links

Important note about browser local data

  • Bot Token and Chat ID are stored in the browser's localStorage so you can reuse them more quickly.
  • Uploaded file history and generated secure links are also stored locally on the current device.
  • If you are using a shared or public computer, click Clear Local Data after you finish.
  • This local data is not encrypted at the frontend layer, so it should not be used on public or untrusted devices.
