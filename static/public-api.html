Public Integration

Use path-based public endpoints without Authorization headers

Integrate TeleRealm with a direct URL pattern such as /link/{botToken}/{chatID} for upload, list, get, update, and delete record operations.

Open Documentation View Demo Benefits

Base Pattern

POST   /link/{botToken}/{chatID}
GET    /link/{botToken}/{chatID}
GET    /link/{botToken}/{chatID}/{recordID}
PATCH  /link/{botToken}/{chatID}/{recordID}
DELETE /link/{botToken}/{chatID}/{recordID}

Integration Flow

How users integrate the public API

1. Prepare bot token

Create a bot with BotFather and copy the token into the path parameter.

2. Choose chat ID

Use your private chat, group, or channel ID where the bot can post files.

3. Call CRUD endpoints

Upload files, read records, update metadata, and delete records by record ID.
